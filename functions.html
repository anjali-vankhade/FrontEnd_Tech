Topic 6 : Functions in JS

Functions are the block of code that performs a specific task, and can be invoked whenever and where ever needed/required.

  • Function definition
  • Function call

Notes - Function parameter are like local variables which will work till the block scope of functions.

Sub Topic 1 : Arrow Functions : (=>)

It is a compact way of writing a function.

eg - const functionName = (param1, param2,...)=>{ // yoour code}

Sub Topic 2 : forEach loop in Array : arr.forEach(callBackFunction)

callBackFunction: A callBackFunction is a function which is passed as an argument to another function.

Sub Topic 3 : Map method in Array : arr.map()

Map creates a new array with the result of some operation. The value its callback returns are used to form a new array.

Filter method in Array : arr.filter()

Filter method creates a new array of elements that gives true for a condition / filter.

Reduce method in Array : arr.reduce()

It performs some operations and reduces the array to a single value. It returns only a single value.
