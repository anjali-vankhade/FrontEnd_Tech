Topic 1 : Variables and Data Types

VARIABLES -

Variables is the container to store the data.

let, const, var keywords
  • var -> variable can be redeclare and updated. A global Variable.
  • let -> variable cannot be redeclare, but can be updated. A block variable.
  • const -> variable cannot be redeclare and cannot be updated. A block variable

DATATYPES -

  • PRIMITIVE DATA TYPES
  • NON PRIMITIVE DATA TYPES
PRIMITIVE DATA TYPES
  • Number ------ let num = 34;
  • String ------ let char = "abc";
  • Boolean ------ let isPass = true;
  • Undefined ----- let a;
  • Null ------ let xyz = null;
  • BigInt ------ let x = BigInt('Tony sharp')
  • Symbol ------- let sym = Symbol('34')
NON PRIMITIVE DATA TYPES
  • Objects
  • Arrays
  • Functions
