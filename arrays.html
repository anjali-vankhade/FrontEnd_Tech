Array Method

  • push() - add to end
  • pop() - delete from end and return
  • concat() - joins multiple arrays and return the result
  • unshift() - add to start
  • shift() - delete from start and return
  • slice() - returns a piece of array - slice(startIdx, endIdx)
  • splice() - change original array (add, delete, replace) - splice(startIdx, delCount, newEl1, newEl2,....)
